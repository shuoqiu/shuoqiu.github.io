jQuery选择器和事件

jQuery选择器

  1. $(“元素名”).text(“需要写入元素中的内容。”);

    $("p").text("p元素被修改了。");

  2. $(“.元素类名”).text(“需要写入元素中的内容。”);

    $(".class").text("p元素被修改了。");

  3. $(“.元素的ID”).text(“需要写入元素中的内容。”);

    $(".id").text("p元素被修改了。");

注意： jQuery的更多选择器需要从jQuery的API官网上来查看。

jQuery事件

  1. 常用事件方法

    • 点击事件
    • 单击事件
      `
      eg:
      $(“button”).click(function(){
      $(this).hide();
      });
      `
    • 双击事件
      `$(“button”).dblclick(function(){
      $(this).hide();
      })`
    • 鼠标悬停事件
      `$(“button”).mouseenter(function(){
      $(this).hide();
      })`
    • 鼠标移开事件
      `$(“button”).mouseleave(function(){
      $(this).hide();
      })`
    • 获取焦点事件
    • 丢失焦点事件

  2. 绑定事件

    绑定一个事件，减少代码执行对电脑内存的消耗；

    `eg:
$(document).ready(function(){
    $("#clickBtn").bind("click", actionName);
});
function actionName(e) {
    console.log("已经绑定了。");
}
给按钮绑定点击事件
`

    可以使用on方法来代替bind方法，官方推荐使用on方法。

  3. 解除绑定事件

    `eg:
$("#clickBtn").unbind("click", actionName);   为按钮解除绑定点击事件
`

    可以使用off方法来代替unbind方法，官方推荐使用off方法。

  4. 事件的目标

  5. 事件的冒泡

    • event.stopPropagation(); 阻止父级的冒泡事件
    • event.stopImmediatePropagation(); 阻止所有的冒泡事件
